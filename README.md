Developpeurs du projet: vduforest et raphd-bee

Dans ce projet nous avons fait 2-3 vues permettant de comprendre le fonctionnement d'une application mobile.
Nous avons un changement de vues au fur et à mesure de la réservation d'un terrain par un client.
Tout d'abord, nous avons une prmière page d'accueil avec les différents sport disponibles.
Ensuite, le client choisit ensuite un sport à pratiquer qui l'envoie sur une page avec les différents horaires disponibles.
Enfin, le client choisit un créneau, passe au paiement avec un récapitulatif du panier contenant le créneau du sport ainsi que le nombre de participants.
